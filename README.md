# 🎬Netflix Clone

A responsive clone of the Netflix landing page built using HTML, CSS, and JavaScript.

## 📁Features

- Responsive design
- FAQ accordion with toggle
- Plus icon transforms into an X on click
- Clean layout similar to Netflix

## Tech Stack

- HTML
- CSS
- JavaScript
